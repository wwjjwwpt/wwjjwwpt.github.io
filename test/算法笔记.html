应用：124 二叉树最大路径![](/Users/wenjiawei/Library/Application Support/typora-user-images/截屏2021-05-17 15.13.41.png)
# Definition for a binary tree node.
# class TreeNode:
#     def __init__(self, val=0, left=None, right=None):
#         self.val = val
#         self.left = left
#         self.right = right
class Solution:
    def __init__(self):
            self.ans = float("-inf")
    def maxPathSum(self, root: TreeNode) -> int:
        def traverse(N):
            if(N == None) : return 0
            left = max(0,traverse(N.left))
            right = max(0,traverse(N.right))
            self.ans = max(self.ans, left + right + N.val)
            return max(left,right)+N.val
        traverse(root)
        return self.ans

解题思路：利用树的遍历算法每一次计算一个小子树的路径总和每一步进行一次ans计算，俺说计算分为左子树和右子树存在的话进行递归计算。

class Solution:
    def buildTree(self, preorder: List[int], inorder: List[int]) -> TreeNode:
        if(len(preorder)==0):
            return None
        if(len(preorder)==1):
            return TreeNode(preorder[0])
        else:
            T = TreeNode(preorder[0])
            T.left = self.buildTree(preorder[1:inorder.index(preorder[0])+1],inorder[:inorder.index(preorder[0])])
            T.right = self.buildTree(preorder[inorder.index(preorder[0])+1:],inorder[inorder.index(preorder[0])+1:])
        return T

在树的解法中涉及递归的算法，而对称性递归在特色算法中的应用也很广泛，特别是在对称的数据结构中，利用递归的的思想快速在数据结构的两端进行条件判断和查找。可以用对称性递归解决的二叉树问题大多是判断性问题(bool类型函数),这一类问题又可以分为以下两类：
